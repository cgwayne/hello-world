The Coordinated Accounting Response (CAR) 2.10 release is now available. This release of CAR 2.10 is primarily a bug fix release.

You can download the latest CAR release from the Project Coordinated Accounting Response Downloads page.

See the Coordinated Accounting Response 2.10 Release Notes for a description of the Known Issues and the Fixed Issues included in this release.

See the What's New In Coordinated Accounting Response page for a description of the important changes in this release.

CAR is a commercial product available as part of Wixard Project Standard Edition (SE) Advanced and Wixard Project SE Suite. CAR helps you manage the use of Project versions and Project applications in your enterprise.

Key benefits of using CAR 2.10 include:

  • Usage Tracking: The Coordinated Accounting Response collects information about which Project applications are launched by which Project versions, creating an application compatibility inventory. Usage Tracking can be configured automatically through the Windows Agent, or by using the configuration properties on other platforms.
  • Taking Action: The Deployment Rule Set capabilities enable different Applet and Web Start applications to use different versions of Project installed on clients. This benefits organizations that must use older versions of Project or need many different Project versions for different reasons.
  • MSI Customizations: System Administrators can use the Project Install Customizer tool to repackage MSIs, controlling the way that Project is installed on managed systems.
